## 영업담당 모드 UI

### 🗂️ 사이드바 메뉴 구조
```
📊 대시보드
📝 실적보고서 작성
✅ 실적보고서 확인
   └─ 📰 고객소식 작성 ⬅️ 새로 추가
   └─ 💬 관리자의견 확인 ⬅️ 새로 추가
🏢 거래처 관리
```

#### 🎨 주요 기능
- **자동완성**: 거래처명 입력 시 기존 거래처 목록에서 자동완성
- **기간 필터**: Date Picker 컴포넌트 사용
- **카테고리 필터**: 다중 선택 가능한 버튼 그룹
- **아이콘**: 카테고리별 시각적 구분 (📌경조사, 🎂생일, 🏢개업기념일, 📰일반)
- **의견 카운트**: 관리자 의견 개수 표시

```
│  [상세 내용]                                                      │

│  │ * 내용:                                                    │   │
│  │ ┌────────────────────────────────────────────────────┐   │   │
│  │ │                                                       │   │   │
│  │ │  (여기에 고객소식 내용을 입력하세요)                   │   │   │
│  │ │                                                       │   │   │
│  │ │                                                       │   │   │
│  │ │                                                       │   │   │
│  │ └────────────────────────────────────────────────────┘   │   │
│  │                                                            │   │
│  │ 💡 작성 아이디어 (클릭하여 템플릿 적용):                   │   │
│  │ • [경조사: 결혼] [경조사: 부고] [생일 축하]                │   │
│  │ • [개업/이전] [수상/인증] [신제품 구매]                    │   │
│  └──────────────────────────────────────────────────────────┘   │
│                                                                  │
│  [알림 설정]                                                      │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ ☑ 로그인 시 전 직원에게 알림 표시                          │   │
│  │                                                            │   │
│  │ 알림 기간: [시작일 📅] 2025-10-15                          │   │
│  │           ~ [종료일 📅] 2025-10-25                         │   │
│  │                                                            │   │
│  │ 중요도: ○ 낮음  ● 보통  ○ 높음  ○ 긴급                    │   │
│  │                                                            │   │
│  │ ℹ️ 알림은 로그인 시 최대 3회까지 표시됩니다                │   │
│  └──────────────────────────────────────────────────────────┘   │
│                                                                  │
│                               [💾 저장하기]  [❌ 취소]           │
└─────────────────────────────────────────────────────────────────┘
```

#### 🎨 주요 기능
- **필수 입력 필드**: * 표시로 구분
- **거래처 자동완성**: 타이핑 시 실시간 검색
- **템플릿 버튼**: 클릭 시 카테고리별 미리 작성된 텍스트 적용
- **매년 반복**: 체크박스로 간단하게 설정
- **알림 설정**: 선택적으로 로그인 알림 활성화
- **중요도 설정**: 4단계 (낮음/보통/높음/긴급)

```
       │  │                                      │  │
       │  │    [상세보기]  [더이상 보지 않기]   │  │
       │  └─────────────────────────────────────┘  │
       │                                            │
       │  ℹ️ 각 소식은 로그인 시 최대 3회까지    │
       │     표시됩니다. (현재: 1/3)             │
       │                                            │
       │              [모두 닫기]                   │
       └───────────────────────────────────────────┘
```

#### 🎨 동작 방식
1. **로그인 시 자동 표시**: 알림 설정된 고객소식 중 다음 조건을 만족하는 것만 표시
   - `showAsNotification = TRUE`
   - 현재 날짜가 `notificationStartDate`와 `notificationEndDate` 사이
   - 해당 직원의 `viewCount < 3` AND `isDismissed = FALSE`

2. **더이상 보지 않기**: 클릭 시 `isDismissed = TRUE` 설정, 이후 표시 안 함

3. **자동 증가**: 팝업 표시될 때마다 `viewCount` 자동 증가

4. **우선순위 정렬**:
   - 긴급 > 높음 > 보통 > 낮음
   - 같은 우선순위는 소식일이 가까운 순

## 💡 추가 기능 아이디어

### 1. 통계 대시보드
```
- 카테고리별 고객소식 통계
- 월별 작성 추이 그래프
- 담당자별 작성 현황
- 중요 소식 미확인 현황
```

### 2. 캘린더 뷰
```
- 월간/주간 캘린더에 고객소식 표시
- 생일, 기념일 등 반복 일정 자동 표시
- 클릭 시 상세 내용 팝업
```

### 3. 검색 기능 강화
```
- 전체 텍스트 검색 (제목 + 내용)
- 고급 검색 (작성자, 카테고리, 기간, 중요도 조합)
- 최근 검색어 저장
```

### 4. 엑셀 내보내기
```
- 선택한 고객소식을 엑셀로 다운로드
- 관리자 의견 포함 여부 선택
- 월간/분기별 리포트 생성
```

### 5. 이메일 알림
```
- 중요 고객소식 등록 시 관련 직원에게 이메일 발송
- 관리자 의견 작성 시 영업담당자에게 알림
- 일일/주간 다이제스트 이메일
```
