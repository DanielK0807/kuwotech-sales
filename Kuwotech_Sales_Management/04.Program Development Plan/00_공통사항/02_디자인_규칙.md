## 1. 디자인 철학

### 1.1 핵심 원칙

#### 🎯 **역할별 차별화 (Role-Based Differentiation)**

```
로그인 → 신뢰와 전문성 → 네이비 + 옐로우
영업담당 → 혁신과 활력 → 시더 + 오렌지  
관리자 → 평온과 트렌디 → 블루 + 민트
```

#### 💎 **글래스모피즘 일관성 (Glassmorphism Consistency)**

```css
/* 모든 테마 공통 글래스 효과 */
.glass-base {
  backdrop-filter: blur(12px);
  background: rgba(var(--theme-rgb), 0.15);
  border: 1px solid rgba(255, 255, 255, 0.18);
  box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.37);
}
```

#### 🎨 **동적 테마 시스템 (Dynamic Theme System)**

```javascript
// 런타임 테마 변경
document.documentElement.setAttribute('data-mode', userRole);
document.body.className = `theme-${userRole}`;
```

### 1.2 디자인 우선순위

1. **가독성** - WCAG 2.1 AAA 기준 충족
2. **일관성** - 모든 컴포넌트 동일한 디자인 언어
3. **반응성** - 모든 디바이스 최적화
4. **성능** - 60fps 애니메이션 유지
5. **접근성** - 키보드/스크린리더 완벽 지원

### 10.3 테마 적용 방법

```javascript
// 테마 초기화
import { ThemeManager } from './01.common/10_theme_manager.js';

const theme = new ThemeManager();

// 로그인 시 역할별 테마 설정
function applyUserTheme(userRole) {
  switch(userRole) {
    case '영업담당':
      theme.setTheme('sales');
      break;
    case '관리자':
      theme.setTheme('admin');
      break;
    default:
      theme.setTheme('login');
  }
}

// 테마 전환 이벤트
document.addEventListener('themeChanged', (e) => {
  console.log(`테마 변경: ${e.detail.theme}`);
});
```
