## 🔍 브라우저 테스트 체크리스트

### 개발자 도구 콘솔 명령어

```javascript
// 1. 테마 관련 테스트
themeManager.getCurrentTheme()           // 현재 테마 확인
themeManager.getThemeList()              // 사용 가능한 테마 목록
themeManager.applyTheme('admin')         // 테마 직접 변경
themeManager.cycleTheme()                // 테마 순환

// 2. CSS 변수 확인
getComputedStyle(document.documentElement).getPropertyValue('--primary-color')
getComputedStyle(document.documentElement).getPropertyValue('--accent-color')
getComputedStyle(document.documentElement).getPropertyValue('--glass-bg')

// 3. 반응형 테스트
breakpointManager.getCurrent()           // 현재 브레이크포인트
breakpointManager.getDeviceType()        // 디바이스 타입
breakpointManager.matches('>=md')        // 미디어쿼리 테스트

// 4. 포맷팅 테스트
window.Format.currency(1234567)          // ₩1,234,567
window.Format.dateKorean(new Date())     // 2025년 1월 27일
window.Format.phone('01012345678')       // 010-1234-5678

// 5. 토스트 테스트
Toast.success('성공!')
Toast.error('오류 발생')
Toast.warning('주의 필요')
Toast.info('정보 메시지')
```

## 📱 반응형 테스트 시나리오

### 1. 모바일 (320px ~ 639px)
- [ ] 사이드바가 오버레이로 전환
- [ ] 테이블이 카드뷰로 변경
- [ ] KPI 카드가 1열로 정렬
- [ ] 글래스 blur 효과 감소

### 2. 태블릿 (640px ~ 1023px)
- [ ] 사이드바 너비 200px
- [ ] KPI 카드가 2열로 정렬
- [ ] 테이블 우선순위 컬럼만 표시

### 3. 데스크톱 (1024px 이상)
- [ ] 사이드바 너비 250px
- [ ] KPI 카드가 3~4열로 정렬
- [ ] 테이블 모든 컬럼 표시
- [ ] 글래스 효과 전체 적용
